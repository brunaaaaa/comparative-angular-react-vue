## BMI Calculator

### Simple project designed in Angular, React and Vue for purposes of comparative study.

<hr>

#### **Angular**

Runs on port `4200`

```
cd angular
npm install
ng serve
```

#### **React**

Runs on port `3000`

```
cd reactjs
npm install
npm start
```

#### **Vue**

Runs on port `8080`

```
cd vuejs
yarn
yarn serve
```
